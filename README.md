# Local-LORA-Gallery-Creator
Python script that creates a local gallery to browse your LORA folder with image previews, trigger words, and filters.

I download all of my Loras off of Civitai using the Firefox Civit Model Downloader https://addons.mozilla.org/en-US/firefox/addon/civit-model-downloader/. This extension saves folders with meta data + image previews with the model. If you put these folders straight into your Lora folder for Stable Diiffusion, this script can be ran from the root folder and it will generate an HTML document that shows all of the files with trigger words, image previews, and filters so that you can easily filter and scroll through your loras outside of your preffered SD program and select the styles you want to use.
